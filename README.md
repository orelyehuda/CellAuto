## Getting Started



### Prerequisites

* C99
* GNU



### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/orelyehuda/CellAuto.git
   ```

## Usage
 ```sh
   gcc cell.c
```
   ```sh
   ./a.out
   ```
   or 
   
   ```sh
   ./a.out '# of generations"
   ```
